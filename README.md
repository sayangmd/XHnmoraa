# 项目简介
## 核心功能
- 前后端代码与SQL文件
- 完整的文档指南
- 参考导入调试视频、讲解视频及开发工具安装包
- 多角色登录管理，包括管理员、商家和用户
## 适用场景
- 美食爱好者分享食谱与心得
- 美食商家推广自家菜式
- 平台管理方维护平台内容
## 技术特点
- 技术架构采用B/S、MVC模式，结合Web框架
- 开发环境配置为IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈为Java、SpringBoot、Vue，提供高效的开发与运行效率
